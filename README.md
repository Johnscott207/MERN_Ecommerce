# MERN_Ecommerce
MERN Based Ecommerce Site


<h3>Features</h3>
1. Products List <br>
2. Add to Cart / Remove from Cart<br>
3. Buy Now (Invoice)<br>
4. Login & Create New Account <br>
5. Admin Panel<br>
6. Add Products Panel<br>

<br>
<br>
<h3>Using</h3>
Frontend: ReactjS<br>
Backend: Express JS<br>
Database: Mongo DB<br>
